# 플루트 핑거링 랩

플루트 초보자가 계이름, 음이름, 오선보, 운지, 실제 소리를 한 화면에서 연결해 배우는 모바일 우선 웹앱입니다. 기준 데이터는 `플룻_운지법_배우기_웹앱_바이브코딩_프롬프트_통합본.md`의 A-9 운지표이며, 앱은 이 데이터를 단일 정답 출처로 사용합니다.

## 실행

```bash
npm install
npm run dev
```

## 검증

```bash
npm run test:run
npm run build
```

## 포함 기능

- 홈: 오늘의 5음, 검색, 첫 방문 3스텝 안내
- 운지 탐색: 음역/표기/도법/조 선택, 플루트 SVG, 대체 운지, 단계별 키 강조, 역방향 운지 찾기
- 오디오: 첫 제스처 언락, smplr 플루트 시도, Web Audio 합성 폴백, 음소거/음량
- 오선보: VexFlow 렌더링과 모바일 탭 슬롯
- 배우기: 키 이름, 음이름과 계이름, 옥타브 개념
- 연습: 스케일과 동요, BPM, 한 음씩 진행, 운지/오선보 동기화
- 퀴즈: 운지→이름, 이름→운지, 오선보→이름, 스트릭과 복습 요약
- 설정: C풋/B풋, 초보자 모드, 모션 줄이기, 햅틱, 음소거, 음량, 데이터 초기화
- 마이크 피치 확인: 로컬 처리만 사용하며, 권한 거부 시에도 앱은 정상 동작

## 데이터 기준

표준 Boehm 시스템, 폐쇄 G♯, C 플루트를 기준으로 합니다. B3와 C7의 기즈모 변형은 B풋 설정에서만 노출됩니다. D4와 D5는 E♭ 키를 떼며, B♭4/B♭5는 엄지 B♭ 레버와 1+1 운지를 모두 제공합니다.

## 주의

일부 고음역 운지와 실제 소리내기는 악기, 손 습관, 교사 지도에 따라 달라질 수 있습니다. 앱은 제공된 기준 데이터를 학습용으로 보여 주며, 장기 연습에서는 선생님 확인을 권장합니다.
